# Film Discovery Web Application

## How to run locally

1. Create and activate a virtual environment:

```bash
python -m venv venv
# Windows (PowerShell)
.\\venv\\Scripts\\Activate.ps1
# macOS / Linux
source venv/bin/activate
```

2. Install dependencies:

```bash
pip install -r requirements.txt
```

3. Initialize the database (optional for first run):

```bash
python scripts/seed_database.py
```

4. Run the application:

```bash
export FLASK_APP=app.py       # or set on Windows: set FLASK_APP=app.py
flask run
# Visit http://127.0.0.1:5000 in your browser
```

## Deployment

This project was deployed to PythonAnywhere for demonstration. Deployment notes are included in `scripts/deploy.py` and `deploy/DEPLOYMENT_README.md` (generated by the `scripts/deploy.py` helper).

## Notes

- The `requirements.txt` file lists required Python packages for reproducibility.  
- Database schema and sample data are provided via scripts in the `scripts/` directory.  
- Do not include virtual environments, compiled caches, or runtime databases in submissions.
